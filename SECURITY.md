# Security policy

## Supported versions

Security fixes are applied to the current `master` branch. LabelDesk has not
yet published a stable release line with long-term support.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting feature for this
repository. If that feature is not available, contact the maintainer privately
at `keralots@gmail.com`.

Include the affected version or commit, reproduction steps, expected impact,
and any suggested mitigation. Do not include sensitive label data, printer
identifiers, or personal information in a report.

Please allow reasonable time for investigation and a fix before public
disclosure. Ordinary bugs and feature requests can use GitHub Issues.
